# dotfiles
![Screenshot of desktop](https://github.com/SimonBurdall/dotfiles/blob/main/preview.png)

The following is a NixOS + Home Manage, flake-based, multi-host setup. The main point of which is to allow for a clean recovery onto a new machine.

## Secrets
The following will need to recreated manually.

 Path                          | What                          |
|-------------------------------|-------------------------------|
| `config/ags/todoist-token`    | Todoist API token for the bar |
